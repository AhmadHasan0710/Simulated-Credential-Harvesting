# Ethical Hacking & Network Defense Configuration
This section contains all the configuration & setup files utilized within this Lab enviorment. 

That would include but aren't limited too:
<ul>
<li>Phishing simulation setup (credential harvesting environment)
<li>IDS/IPS configuration for detecting malicious activity
<li>Network analyzer setup for monitoring traffic
<li>Firewall rules and segmentation
<li>Supporting tools used across the lab
</ul>

This setup is designed to show not only how oppurtunities are used maliciously by attackers to potentially steal credentials, but how many interpreise level networks not only detect these attacks, but mitigate any form of attack.
