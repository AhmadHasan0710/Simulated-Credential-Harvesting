<h1>Ethical Hacking Tools</h1>

<p>
This folder contains the tools and configurations used for the offensive side of this lab. These tools are used in a controlled environment to simulate real-world attack scenarios, only for educational and defensive analysis purposes.
</p>

<p>
The focus of this section is phishing simulation and credential harvesting, which is then analyzed and detected using the network defense tools configured in the rest of the lab, on how many networks infrastructures would not only detect, but minigate such threats.
</p>

<p>
Included in this section are configurations for tools such as:
</p>

<ul>
<li>GoPhish (phishing campaign framework for email and credential capture simulation)</li>
<li>Social-Engineer Toolkit (SET) (toolkit for creating phishing pages and attack simulations)</li>
</ul>

<p>
Within real world scenerios, many attackers would Typically use far more advanced techniques/tools in order to achieve their own malicious objectives. This lab is primarly focused on the network's response on any form of credential harvesting, rather then the tehcnique/tools themselves.
</p>
