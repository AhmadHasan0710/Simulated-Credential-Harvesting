# Simulated-Credential-Harvesting
A simulated, targeted phishing attack lab built on a segmented network, designated to demonstrate not only credential harvesting techniques, but real time netowrk analyzatoin, detection, & mitigation. This project utilizes tools/Firewalls such as pfSense, pfBlockerNG-devel, Suricata, Zeek, Darkstat, and etc; while utilizing GoPhish, SET, reverse proxy, and etc for offensive simulation. The network architecture used within this project mirrors the [Network Infrastructure Homelab](https://github.com/AhmadHasan0710/Network-Infrastrucutre-Homelab), emphasizing VLAN segmentation, traffic analysis, threat detection, and active response to any and all forms of malicious activity.
